# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-18

### Added

- Initial project setup with complete directory structure
- Arabic text preprocessing pipeline
- Semantic search using FAISS embeddings
- Arabic query processing with normalization
- RAG chain for question answering
- FastAPI backend with REST API
- Answer generation with citations
- Configuration management via YAML
- Comprehensive test suite
- Demo script for local testing
- GitHub Actions CI/CD pipeline
- Documentation and contribution guidelines

### Features

- Multi-madhhab Fiqh knowledge base support
- Arabic-native text processing
- Flexible embedding model selection
- Configurable retrieval parameters
- Source attribution and citation tracking
- Interactive API with Swagger documentation
- Extensible architecture for new sources and models

### Technical Stack

- **Framework**: LangChain, FastAPI
- **Vector Store**: FAISS
- **Embeddings**: Sentence Transformers
- **Arabic NLP**: PyArabic, CAMeL Tools
- **Testing**: Pytest
